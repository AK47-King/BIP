    铜梁区交通局是主管全区城乡公共客运交通、水路交通的行政管理和行业
管理的工作部门。负责全区公路、水路及其设施的规划、设计、招投标、建
设、工程监理、质量监督、养护管理及路政、运政管理，公路、水路建设市
场管理和规费征收管理。负责全区范围内的路政、运政、交通运输安全，交
通运输秩序的综合执法管理。负责全区道路、水路运输的行业管理；负责规
划、协调和审批全区道路客货运输线路、运输站（场、点）、码头航道的建
设、地方船舶运输管理及运输市场的设置，根据市场需要管理全区营运性运
输车辆、船舶的投放；负责运输企业、汽车维修市场、 车辆技术及运输船
舶制造维修的行业管理。
